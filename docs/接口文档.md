# 接口文档

后端默认地址：`http://localhost:8000`。启动后也可访问 `/docs` 查看 Swagger。

## 1. 教材

### POST `/api/textbooks/upload`

上传多个教材文件。

请求：`multipart/form-data`

字段：`files: File[]`

返回：

```json
[
  {
    "textbook_id": "book_xxx",
    "filename": "生理学.pdf",
    "title": "生理学",
    "format": "pdf",
    "total_pages": 449,
    "total_chars": 800000,
    "status": "已完成",
    "chapters": []
  }
]
```

### GET `/api/textbooks`

返回已解析教材列表。

## 2. 知识图谱

### POST `/api/graph/{textbook_id}/build`

构建单本教材知识图谱。

返回：

```json
{
  "textbook_id": "book_xxx",
  "nodes": [
    {
      "id": "node_001",
      "name": "动作电位",
      "definition": "...",
      "category": "核心概念",
      "chapter": "第二章 细胞的基本功能",
      "page": 31,
      "textbook_id": "book_xxx",
      "textbook_title": "生理学"
    }
  ],
  "edges": [
    {
      "source": "node_001",
      "target": "node_002",
      "relation_type": "prerequisite",
      "description": "理解动作电位需要先掌握静息电位"
    }
  ]
}
```

## 3. 跨教材整合

### POST `/api/integration/run`

自动补建缺失图谱并整合。

返回：`IntegratedGraph`

```json
{
  "nodes": [],
  "edges": [],
  "decisions": [
    {
      "decision_id": "merge_001",
      "action": "merge",
      "affected_nodes": ["book01_node_015", "book03_node_032"],
      "result_node": "merged_node_001",
      "reason": "语义高度相似...",
      "confidence": 0.92
    }
  ],
  "stats": {
    "original_chars": 1000000,
    "integrated_chars": 280000,
    "compression_ratio": 28.0
  }
}
```

## 4. RAG

### POST `/api/rag/index`

建立知识库索引。

### GET `/api/rag/status`

查看索引状态。

### POST `/api/rag/query`

请求：

```json
{
  "question": "动作电位是什么？",
  "top_k": 5
}
```

返回：

```json
{
  "answer": "... [生理学, 第二章, 第 31 页]",
  "citations": [
    {
      "textbook": "生理学",
      "chapter": "第二章 细胞的基本功能",
      "page": 31,
      "relevance_score": 0.91,
      "chunk_id": "chunk_xxx"
    }
  ],
  "source_chunks": [
    {
      "chunk_id": "chunk_xxx",
      "text": "原文片段..."
    }
  ],
  "latency_ms": 213
}
```

## 5. 教师对话

### POST `/api/dialogue/message`

请求：

```json
{"message":"我觉得免疫应答不应该被删除，请保留"}
```

返回：

```json
{
  "reply": "已根据教师反馈保留...",
  "updated_decisions": [],
  "graph_changed": true
}
```

## 6. 报告与 Benchmark

- `POST /api/report/generate`
- `GET /api/report/download?format=md`
- `GET /api/report/download?format=pdf`
- `POST /api/benchmark/run`
